# ETHTrading
### Web 3.0 Blockchain App where you can trade ETH.
### This project was bootstrapped with [Vite](https://github.com/vitejs/vite).

- React, JavaScript
- Vite
- TailwindCSS
- Solidity
- Smart Contracts
- Web3
- Blockchain
- MetaMask

- Send ETH to another wallet
- View your transactions history
